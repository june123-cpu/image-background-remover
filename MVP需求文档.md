# Remove Background - MVP 需求文档

## 1. 项目概述

| 项目名称 | Remove Background (AI 智能抠图) |
|---------|-------------------------------|
| 项目类型 | Web 应用 (工具类 SAAS) |
| 核心功能 | 纯前端上传图片，AI 自动移除背景 |
| 目标用户 | 电商卖家、设计师、自媒体创作者 |
| 部署平台 | Cloudflare Pages |

---

## 2. 功能需求

### 2.1 核心功能

| 优先级 | 功能 | 描述 |
|-------|------|------|
| P0 | 图片上传 | 拖拽或点击选择图片 |
| P0 | AI 抠图 | 调用 Remove.bg API 移除背景 |
| P0 | 结果预览 | 前后对比展示 |
| P0 | 图片下载 | 保存透明背景 PNG |
| P1 | 批量处理 | 连续处理多张图片 |
| P1 | 历史记录 | 浏览器本地存储最近 10 张 |
| P2 | 用户系统 | 登录后可查看历史 |

### 2.2 非功能需求

| 需求项 | 描述 |
|-------|------|
| 响应时间 | 页面加载 < 2s，抠图处理 < 10s |
| 图片限制 | 最大 10MB，支持 JPG/PNG/WebP |
| 兼容性 | 现代浏览器 (Chrome/Firefox/Safari/Edge) |
| 可用性 | 纯内存处理，无存储成本 |

---

## 3. 技术方案

### 3.1 技术栈

```
前端:       Next.js 15 + React 18 + Tailwind CSS
后端:       Next.js API Routes
AI 服务:   Remove.bg API
部署:       Cloudflare Pages
域名:       可选配置自定义域名
```

### 3.2 成本估算

| 项目 | 免费额度 | 超出费用 |
|------|---------|---------|
| Remove.bg API | 50次/月 | $0.07/张 |
| Cloudflare Pages | 无限 | 免费 |
| 域名 | 可选 | ~$10/年 |

## 5. API 设计

### 5.1 前端 → 后端

**POST /api/remove-bg**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| image_file | File | 是 | 图片文件 |

**响应 (成功)**
```
Content-Type: image/png
Content-Disposition: attachment; filename="removed-bg.png"
```

**响应 (失败)**
```json
{
  "error": "错误信息"
}
```

## 7. 风险与限制

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| API 配额用尽 | 无法抠图 | 监控用量，提醒用户 |
| 图片太大 | 处理失败 | 前端限制 10MB |
| 网络超时 | 请求失败 | 添加重试机制 |
| Remove.bg 服务不可用 | 功能不可用 | 预留其他 API |
